# Stomatal anatomy metadata

This contains metadata for stomata_anatomy

## Usage

``` r
stomata_anatomy_metadata
```

## Format

A data frame with 38 rows and 7 variables:

- `source`:

  character Source paper identifier.

- `taxa`:

  character Description of the taxa covered in source.

- `location`:

  character Location where trait data are located in source. In some
  cases, raw data were provided directly by the authors.

- `density_unit`:

  character Unit used for stomatal density.

- `length_unit`:

  character Unit used for guard cell or pore length or width.

- `sample_size`:

  double Approximate sample size for each taxon in the study.

- `notes`:

  character Notes to understand units or sample size data.
